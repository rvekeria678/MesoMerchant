How to get Started

MesoMerchant is easy to download and install so that players can get into the anicent world quicker. The game is able to run on a variety of different platforms: Windows, Linux, Mac, Android

Solarus acts similar to running an emulator such that quests are loaded like ROM files. MesoMerchant currently runs on version 1.6.5 of Solarus. Make sure to download the player version of the launcher.
Next, the quest can be downloaded from the team's repository. To the right of "Clone" at the top right, click the three dots and select "Download as Zip".
